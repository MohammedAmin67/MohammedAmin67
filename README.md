### 🌌 Impact Lab — Asteroid Impact Simulator
An interactive asteroid impact simulator and planetary defense game with scientifically accurate orbital mechanics. Visualize real impact energy, crater sizes, and damage predictions — then defend Earth using kinetic impactors, nuclear deflection, or gravity tractors.  
**Stack:** React · Vite · Three.js · D3.js · Tailwind CSS  
[🔴 Live Demo](https://meteor-madness-frontend.onrender.com) · [GitHub Repo](https://github.com/MohammedAmin67/meteor-madness)

### 🎓 EduSmart — AI-Powered Learning Platform
A full-stack edtech platform with dual roles for students and faculty. Features gamified quizzes, XP/badge achievement systems, real-time progress analytics, doubt resolution, and **Synapse** — a built-in AI assistant (powered by Google Gemini) that helps students learn and navigate the platform.  
**Stack:** React · Vite · Node.js · Express · MongoDB · Google Gemini API · Cloudinary  
[🔴 Live Demo](https://edusmart-learn.vercel.app) · [GitHub Repo](https://github.com/MohammedAmin67/edusmart-learning)
